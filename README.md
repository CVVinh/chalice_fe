# Lospec LCM Development

## Tool versions

node js: 16.15.0
yarn: 1.22.19
vuetify: 3.0.2
Vue Router: 4.1.6

Please refer to `yarn.lock` for other package installed.

## How to setup dev environment

1. Clone this repository
2. run `yarn install`

## Project setup

```
# yarn
yarn

# npm
npm install

# pnpm
pnpm install
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev
```

### Compiles and minifies for production

```
# yarn
yarn build

# npm
npm run build

# pnpm
pnpm build
```
